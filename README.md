# ADFincas

Sitio web para una empresa de administración de fincas y servicios inmobiliarios. Desarrollado con React y Vite.

## Stack

- **React 19** + **Vite 7**
- **Tailwind CSS 3**
- **EmailJS** — envío de formularios
- **FontAwesome 7** — iconografía
- **Jest** — testing unitario

## Desarrollo

```bash
npm install
npm run dev        # http://localhost:5173
```

## Producción

```bash
npm run build
npm run preview
```

## Linting

```bash
npm run lint
```

## Estructura

```
src/
├── components/    # Secciones de la página
├── data/          # Constantes y datos de contenido
├── pages/         # Páginas independientes (legal, privacidad)
└── utils/         # Configuración de SEO, analytics y formularios
```
